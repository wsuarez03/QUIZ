- [ ] **Crear cuenta** en https://firebase.google.com (gratis)

## 🟡 FASE 2: CONFIGURACIÓN LOCAL (Tiempo: 5 min)

- [ ] Abrir carpeta `quiz-app` en VS Code
- [ ] Copiar `.env.local.example` → `.env.local`
- [ ] Pegar valores de Firebase en `.env.local`
- [ ] Instalar dependencias: `npm install`
- [ ] Probar localmente: `npm run dev`

## 🔴 FASE 3: DEPLOY (Tiempo: 10 min)

- [ ] Crear repositorio en GitHub
- [ ] Hacer commit y push a main branch
- [ ] Conectar Vercel con GitHub
- [ ] Agregar variables de entorno en Vercel
- [ ] Hacer deploy

## 🟢 FASE 4: VERIFICACIÓN (Tiempo: 5 min)

- [ ] Abrir URL de Vercel en navegador
- [ ] Registrarse con email de prueba
- [ ] Iniciar sesión
- [ ] Navegar por la aplicación
- [ ] ¡Verificar que todo funciona!
